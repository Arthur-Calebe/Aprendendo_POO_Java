# 📚 Aprendendo POO em Java

Este repositório reúne meus estudos sobre **Programação Orientada a Objetos (POO)** utilizando **Java**.

Aqui você encontrará exemplos práticos, exercícios e projetos desenvolvidos ao longo da minha jornada de aprendizado, com base nas aulas da **Maratona Java** do **DevDojo**.

## 🎯 Objetivos

- Aprimorar os fundamentos da Programação Orientada a Objetos.
- Praticar conceitos essenciais do Java.
- Registrar minha evolução como desenvolvedor.
- Criar uma base de consulta para estudos futuros.

## 📖 Conteúdos abordados

- Classes e Objetos
- Métodos
- Construtores
- Encapsulamento
- Herança
- Polimorfismo
- Abstração
- Interfaces
- Sobrescrita e Sobrecarga
- Associação entre objetos
- Modificadores de acesso
